## Welcome to USB Flash Drive Online  ##
开源PHP在线u盘系统，欢迎使用！



### How to install it? ###
* 打开https://github.com/RudonTeam/USB-Flash-Drive-Online
* Clone or Download > Download Zip > 解压到服务器项目目录下，重命名目录为`upan`
* 完成
* 打开https://your-site.com/upan 即可享用
